# Basically Homebrew Method  
## 搜尋套件  
```
brew search 套件名稱
```  

## 搜尋應用程式
```
brew cask search 應用程式名稱
```

## 套件資訊  
```
brew info 套件名稱
```

## 安裝套件  
```
brew install 套件名稱
```  

## 安裝應用程式
```
brew cask install 應用程式名稱
```

## 目前已經安裝的套件  
```
brew list
```  

## 目前已經安裝的應用程式  
```
brew cask list
```  

## 更新 Homebrew  
```
brew update
```  

## 解除安裝套件  
```
brew unintstall 套件名稱
```  

## 解除安裝應用程式  
```
brew unintstall 應用程式名稱
```

## Help
```
brew -h
```
